# Camunda8 - Run

Uruchomienie platformy oraz jej ubijanie jest realizowane poprzez c8run.exe
Uwaga: Nie ubijamy procesów ręcznie (c8run odpowiada za resource disposing).

# Uruchomienie środowiska Camunda8:

Z poziomu powershell (admin) wywołaj: 
```powershell 
.\c8run.exe start
```

## Shutdown środowiska Camunda8: 
Z poziomu powershell (admin) wywołaj: 
```powershell 
.\c8run.exe stop
```

Więcej na ten temat [tutaj](https://docs.camunda.io/docs/next/self-managed/setup/deploy/local/c8run/)
