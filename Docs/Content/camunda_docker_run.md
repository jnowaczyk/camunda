# Uruchomienie środowiska Camunda8:


## Start 
Z poziomu powershell (admin) przejdź do katalogu, gdzie rozpakowano docker compose a następnie wywołaj: 
```powershell 
docker compose up -d
```

## Shutdown:
Z poziomu powershell (admin)  przejdź do katalogu, gdzie rozpakowano docker compose a następnie wywołaj: 
```powershell 
docker compose downShutdown
```
