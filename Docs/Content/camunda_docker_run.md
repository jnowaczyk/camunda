# Camunda8 - docker

Uruchamianie via docker compose [instalacja](Camunda_install_docker.md)


# Uruchomienie środowiska Camunda8:


## Start 
Z poziomu powershell (admin) wywołaj: 
```powershell 
docker compose up -d
```

## Shutdown:
Z poziomu powershell (admin) wywołaj:  
```powershell 
docker compose downShutdown
```
